# Restaurant-Page
### This is a website for a fictional Japanese cuisine restaurant.
The aim of the project was to create a functional multi-page website using Webpack to bundle all files.
## Built with
- JS
- HTML
- CSS
## Live Demo
https://just2102.github.io/Restaurant-Page/ 

## Working with this project

1. Clone this project
2. Install the required dependencies:
`npm install`
3. Distribution files can be produced using:  
`npm run build`
